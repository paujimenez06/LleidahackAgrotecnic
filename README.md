## Instalació

1.  **Clonar el repositori**
2.  **Crear un entorno virtual**:
    ```bash
    python -m venv .venv
    .venv\Scripts\activate
    ```
3.  **Instalar dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

### Execució Automática (Windows)
Si estàs a Windows, pots fer doble clic al fitxer run_simulation.bat.
